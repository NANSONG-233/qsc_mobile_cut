# ZJU Mobile Assistant

一个专为浙江大学学生设计的移动端学习助手应用，帮助你管理课程、任务、成绩和专注时间。

## ✨ 功能特性

### 📅 课程表 (Schedule)
- **日视图/周视图** 切换查看课程安排
- 添加、编辑、删除课程
- 支持自定义上课时间
- 显示课程地点和教师信息
- 多种颜色标签区分不同课程

### ✅ 任务管理 (Tasks)
- 创建待办任务
- 按分类筛选任务（学习、生活等）
- 支持自定义分类标签
- 标记任务完成状态
- 删除已完成或不需要的任务

### ⏱️ 专注模式 (Focus)
- 番茄钟计时器
- 自定义专注时长和休息时长
- 专注/休息模式切换
- 可视化进度显示
- 暂停、继续、重置功能

### 📊 成绩管理 (Grades)
- 记录课程成绩和学分
- 自动计算 GPA（4.0 和 5.0 制）
- 显示加权平均分
- 查看总学分统计

### ⚙️ 设置 (Settings)
- 用户信息管理
- 数据导出（JSON 格式备份）
- 数据导入（恢复备份）
- 本地数据持久化存储

## 🚀 本地运行

**前置条件：** Node.js (v18+)

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 打开浏览器访问 `http://localhost:3000`

## 🛠️ 技术栈

- **React 19** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库

## 📱 移动端适配

本应用专为移动端设计，在桌面端会以手机模拟器形式展示。
